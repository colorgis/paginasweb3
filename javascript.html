JavaScript

Es un lenguaje de programación ligero, interpretado, o compilado justo-a-tiempo (just-in-time) con funciones de primera clase.

Ver más Regresar

Historia

El origen javascript surge a partir de un navegador que actualmente no existe: NetScape. En 1995, Netscape se dio cuenta de que, cuando tenía algún documento HTML, no podía interactuar con lo que hacía el usuario.

Entonces, fue Brendan Eich quien desarrollo javascript para NetScape. Este lenguaje de programación le da muchas más posibilidades a los navegadores.

Usos

1Movimiento

Anteriormente, las páginas web eran estáticas, similares a las páginas de un libro. JavaScript surgió como una tecnología del lado del navegador para hacer que las aplicaciones web fueran más dinámicas.

2Interacción

Por medio de JavaScript, los navegadores eran capaces de responder a la interacción de los usuarios y cambiar la distribución del contenido en la página web.

3Desarrollo

En la actualidad, puede utilizar JavaScript para el desarrollo tanto del lado del cliente como del lado del servidor.

4Efectos

Es un lenguaje utilizado para dotar de efectos y procesos dinámicos e “inteligentes” a documentos HTML.
