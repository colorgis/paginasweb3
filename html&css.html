HTML & CSS

HTML y CSS en realidad no son lenguajes de programación técnica; son solo la estructura de la página y la información de estilo. El HTML es el núcleo de cada página web. HTML significa lenguaje de marcado de hipertexto. "Lenguaje de marcado “el HTML utiliza etiquetas para identificar diferentes tipos de contenido y los fines de cada uno para la página web. CSS no son más que hojas de estilo en cascada.

Historia

El origen de HTML se remonta a 1980, cuando el físico Tim Berners-Lee, investigador del CERN (Organización Europea para la Investigación Nuclear) propuso un nuevo sistema de “hipertexto” para compartir documentos. Tras finalizar el desarrollo de su sistema, Tim Berners-Lee lo presentó a una convocatoria organizada para desarrollar un sistema de hipertexto para internet. Después de unir sus fuerzas con el ingeniero de sistemas Robert Cailliau, presentaron la propuesta ganadora llamada WorldWideWeb (W3). El primer documento formal con la descripción de HTML se publicó en 1991 bajo el nombre “HTML Tags” (Etiquetas HTML) y todavía hoy puede ser consultado en línea a modo de reliquia informática.

La propuesta CHSS fue realizada por Håkon Wium Lie y SSP fue propuesto por Bert Bos. Entre finales de 1994 y 1995 Lie y Bos se unieron para definir un nuevo lenguaje que tomaba lo mejor de cada propuesta y lo llamaron CSS (Cascading Style Sheets). En 1995, el W3C decidió apostar por el desarrollo y estandarización de CSS y lo añadió a su grupo de trabajo de HTML. A finales de 1996, el W3C publicó la primera recomendación oficial, conocida como "CSS nivel 1". A principios de 1997, el W3C decide separar los trabajos del grupo de HTML en tres secciones: el grupo de trabajo de HTML, el grupo de trabajo de DOM y el grupo de trabajo de CSS.

Usos

1Desarrollo de páginas web.

HTML se usa mucho para crear páginas que se muestran en la red mundial. Cada página contiene un conjunto de etiquetas HTML que incluyen hipervínculos que se utilizan para conectarse a otras páginas. Cada página que presenciamos, en la red mundial, está escrita con una versión de código HTML.

2Creación de documentos web.

La creación de documentos en Internet está dominada por HTML y su concepto básico a través de etiqueta y DOM, es decir, modelo de objeto de documento.

3Navegación por internet.

Esta navegación es posible utilizando el concepto de hipertexto. Básicamente es un texto que se refiere a otras páginas web o texto y cuando el usuario hace clic en él, navega al texto o página referenciado.

Usos CSS

1Ordenamiento.

CSS es uno de los lenguajes más importantes que se utilizan para ordenar las instrucciones referentes a la apariencia.

2Diseño.

El CSS se encarga de añadir toda la capa de personalización sobre el que la web define su aspecto final.
